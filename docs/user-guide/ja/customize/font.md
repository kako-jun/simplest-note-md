# カスタムフォント

好きなフォントをアップロードして、アプリ全体に適用できます。

---

## 対応フォーマット

- `.ttf`（TrueType）
- `.otf`（OpenType）
- `.woff` / `.woff2`（Web Font）

---

## アップロード方法

1. 設定画面を開く
2. 「フォント選択」ボタンをタップ
3. フォントファイルを選択
4. 即座に適用（リロード不要）

---

## 削除

「デフォルトに戻す」ボタンで元に戻せます。

---

## 注意

- フォントはブラウザのIndexedDBに保存されます
- GitHubには保存されません（サイズが大きいため）
- 複数デバイスで使う場合は、各デバイスで再アップロードが必要
