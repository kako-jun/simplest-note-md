# テーマ

6種類のテーマから選べます。

---

## テーマ一覧

| テーマ         | 特徴                           |
| -------------- | ------------------------------ |
| **yomi**       | 落ち着いた黄色系（デフォルト） |
| **campus**     | 温かみのあるキャンパスノート風 |
| **greenboard** | 黒板風のダークテーマ           |
| **whiteboard** | 明るいホワイトボード風         |
| **dotsD**      | ドット柄のダークテーマ         |
| **dotsF**      | ドット柄の明るいテーマ         |

---

## 変更方法

1. 設定画面を開く（ヘッダー右上の ⚙️）
2. テーマボタンをタップ
3. 即座に反映されます

---

## アプリアイコン

ヘッダー左上にアプリアイコンが表示されます（ツール名がデフォルトの「Agasteer」の場合のみ）。

- テーマに応じた**アクセントカラー**で描画
- テーマを変えるとアイコンの色も変わります
