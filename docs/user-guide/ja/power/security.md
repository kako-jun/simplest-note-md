# セキュリティ

データの保存場所とセキュリティに関する注意事項です。

---

## データの保存場所

| データ       | 保存先                            |
| ------------ | --------------------------------- |
| GitHub Token | LocalStorage（平文）              |
| ノート内容   | IndexedDB + GitHub（HTTPS暗号化） |
| 設定情報     | LocalStorage                      |

---

## 推奨事項

### やるべきこと

- ✅ 信頼できる個人端末でのみ使用
- ✅ トークンには適切な有効期限を設定
- ✅ 定期的にトークンをローテーション

### やってはいけないこと

- ❌ 共用端末では使用しない
- ❌ トークンを他人と共有しない
