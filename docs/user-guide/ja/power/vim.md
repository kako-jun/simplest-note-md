# Vimモード

CodeMirrorのVimキーバインディングを使用できます。

---

## 有効化

1. 設定画面を開く
2. 「Vimモードを有効にする」をON

---

## 基本操作

### ノーマルモード

| キー                  | 動作                        |
| --------------------- | --------------------------- |
| `i` / `a` / `o`       | 挿入モードへ                |
| `h` / `j` / `k` / `l` | カーソル移動                |
| `dd`                  | 行削除                      |
| `yy`                  | 行コピー                    |
| `p`                   | 貼り付け                    |
| `u`                   | アンドゥ                    |
| `Ctrl-r`              | リドゥ                      |
| `<Space>`             | ペイン切り替え（2ペイン時） |

### Agasteer専用コマンド

| コマンド | 動作                     |
| -------- | ------------------------ |
| `:w`     | GitHubにPush（保存）     |
| `:q`     | 親ノートへ遷移（閉じる） |
| `:wq`    | 保存して閉じる           |

---

## 2ペインでの動作

- 左右それぞれでVimモードが独立して動作
- `<Space>`で左右のペインを切り替え
- 各ペインで独立してコマンド実行可能
