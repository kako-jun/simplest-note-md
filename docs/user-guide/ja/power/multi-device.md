# 複数デバイスでの同期

PC、スマホなど複数デバイスで使う場合の注意点です。

---

## 基本的な流れ

1. **デバイスA**で編集 → Push
2. **デバイスB**でPull → 編集 → Push
3. **デバイスA**でPull → ...

---

## 注意

- **競合解決UIはまだ実装されていない**
- 複数デバイスで同時編集すると、後からPushしたデータで上書き
- 必ず編集前にPullを実行

---

## Stale警告

別デバイスで編集された場合、Push前に警告が表示されます。

- 上書きするか、先にPullするか選べます
- 5分間隔でバックグラウンドチェックも行われます
