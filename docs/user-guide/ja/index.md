# Agasteer ユーザーガイド

> 「こういうのでいいんだよ」を志向するMarkdownノートアプリ

---

## はじめに

### [1. クイックスタート](./quick-start.md)

5分でAgasteerを使い始めましょう。

- デモサイトを開く
- GitHub連携を設定する
- 最初のノートを作る
- 最初のリーフを作る

### [2. GitHub連携セットアップ](./github-setup.md)

ノートを作成するために必要なGitHub連携の詳しい手順。

- リポジトリの作成
- トークンの取得
- Agasteerでの設定

## 使い方

### [3. 基本操作](./basic-usage.md)

ノートとリーフの作成・編集・整理の方法。

- ノートの操作（作成、編集、並び替え、削除）
- リーフの操作（作成、編集、並び替え、移動、削除）
- 検索
- Push/Pull

### [4. カスタマイズ](./customization.md)

Agasteerの見た目を自分好みに変えましょう。

- テーマ（6種類）
- 罫線モード
- カスタムフォント
- カスタム背景画像
- 言語設定

### [5. 便利な機能](./features.md)

知っておくと便利な機能。

- 2ペイン表示
- マークダウンプレビュー
- 画像ダウンロード・シェア
- アーカイブ
- インポート/エクスポート
- PWA対応

### [6. パワーユーザー向け](./power-user.md)

上級者向けの機能。

- Vimモード
- Priorityリーフ
- Offlineリーフ
- GitHub連携の詳細
- セルフホスト
