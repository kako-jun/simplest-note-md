# カスタマイズ

> Agasteerの外観をカスタマイズする方法を説明します。

---

## 🎨 テーマの選択

Agasteerは6種類のテーマを用意しています。

### テーマ一覧

1. **yomi** - シンプルで読みやすいデフォルトテーマ
   - 明るい背景に黒いテキスト
   - 長時間の読書に最適

2. **campus** - 落ち着いた色合いのキャンパステーマ
   - 温かみのある色調
   - 勉強や仕事に集中できるデザイン

3. **greenboard** - 緑色の黒板風デザイン
   - 懐かしい黒板の雰囲気
   - 白いチョークのようなテキスト

4. **whiteboard** - 明るいホワイトボード風デザイン
   - クリーンで明るい印象
   - プレゼン資料の作成に最適

5. **dotsD** - ドット柄のダークテーマ
   - 目に優しいダークモード
   - 夜間の作業に最適

6. **dotsF** - ドット柄の明るいテーマ
   - ポップなドット柄
   - カジュアルな雰囲気

### テーマの変更方法

1. 設定画面を開く（ヘッダー右上の⚙️アイコン）
2. テーマセクションでボタンをクリック
3. 即座にテーマが適用されます
4. 💾ボタン（保存）をクリックして設定を保存

### テーマの保存先

- ブラウザのLocalStorageに保存されます
- GitHub連携を設定している場合、`metadata.json`にも保存されます

## 🔤 カスタムフォントのアップロード

好きなフォントをアップロードして、アプリ全体に適用できます。

### 対応フォーマット

- `.ttf` (TrueType Font)
- `.otf` (OpenType Font)
- `.woff` (Web Open Font Format)
- `.woff2` (Web Open Font Format 2)

### フォントのアップロード方法

1. 設定画面を開く
2. カスタムフォントセクションで「フォント選択」ボタンをクリック
3. フォントファイルを選択
4. 即座にアプリ全体に適用されます（リロード不要）
5. 💾ボタン（保存）をクリックして設定を保存

### フォントの削除

1. 設定画面を開く
2. 「デフォルトに戻す」ボタンをクリック
3. 元のフォントに戻ります

### フォントの保存先

- ブラウザのIndexedDB（`fonts`オブジェクトストア）に保存されます
- Base64エンコードされて保存されます
- GitHubには保存されません（サイズが大きいため）

### フォントの注意事項

- フォントファイルのサイズが大きい場合、アップロードに時間がかかります
- 日本語フォントは特にサイズが大きいため、注意が必要です
- ブラウザのIndexedDBの容量制限に注意してください

## 🖼️ カスタム背景画像の設定

左右のペイン別々に背景画像を設定できます。

### 対応フォーマット

- `.jpg` / `.jpeg` (JPEG)
- `.png` (PNG)
- `.webp` (WebP)
- `.gif` (GIF)

### 背景画像のアップロード方法

1. 設定画面を開く
2. カスタム背景画像セクションで「背景画像選択（左）」または「背景画像選択（右）」ボタンをクリック
3. 画像ファイルを選択
4. 即座に背景画像が適用されます

### 背景画像の表示

- 透明度0.1で半透明表示されます
- テキストの可読性を損なわないように調整されています
- 画像は画面全体に表示されます（`background-size: cover`）

### 背景画像の削除

1. 設定画面を開く
2. 「デフォルトに戻す（左）」または「デフォルトに戻す（右）」ボタンをクリック
3. 背景画像が削除されます

### 背景画像の保存先

- ブラウザのIndexedDB（`backgrounds`オブジェクトストア）に保存されます
- Base64エンコードされて保存されます
- GitHubには保存されません（サイズが大きいため）

### 背景画像の注意事項

- 画像ファイルのサイズが大きい場合、アップロードに時間がかかります
- ブラウザのIndexedDBの容量制限に注意してください
- 左右で異なる背景画像を設定できます

## 📝 罫線エディタモード

エディタに罫線（横線）を表示して、紙のノートのような見た目にできます。

### 罫線モードの有効化

1. 設定画面を開く
2. 「罫線モード」をONに切り替え
3. エディタに罫線が表示されます

### 特徴

- 各行に罫線が表示され、視認性が向上
- ライトテーマ・ダークテーマ両方に対応した線色
- 行番号も一緒に表示
- 設定はLocalStorageに保存されます

## 🌍 言語設定

Agasteerは日本語と英語に対応しています。

### 対応言語

- **日本語** (ja)
- **English** (en)

### 言語の自動検出

初回起動時、ブラウザの言語設定を自動検出して適用されます。

- ブラウザが日本語設定の場合 → 日本語で表示
- それ以外 → 英語で表示

### 言語の手動切替

1. 設定画面を開く
2. 言語セクションで「日本語」または「English」を選択
3. 即座に言語が切り替わります
4. 💾ボタン（保存）をクリックして設定を保存

### 言語の保存先

- ブラウザのLocalStorageに保存されます
- GitHub連携を設定している場合、`metadata.json`にも保存されます

---

## 💾 設定の保存

すべてのカスタマイズ設定は以下の方法で保存されます。

### LocalStorageに保存される設定

- GitHub Token
- リポジトリ名
- コミットユーザー名・メールアドレス
- テーマ
- 言語設定
- Vimモード

### IndexedDBに保存される設定

- カスタムフォント（Base64）
- カスタム背景画像（左右別々、Base64）
- ノート・リーフのデータ

### GitHubに保存される設定

- テーマ
- 言語設定
- Push回数カウント

これらは`metadata.json`に保存されます。
