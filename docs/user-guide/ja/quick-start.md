# クイックスタート

> 5分でAgasteerを使い始めましょう。

---

## 1. デモサイトを開く

**[https://agasteer.llll-ll.com](https://agasteer.llll-ll.com)** にアクセスするだけで、すぐに使い始められます。

- インストール不要
- アカウント登録不要
- ブラウザで開くだけ

初回アクセス時に**ウェルカムポップアップ**が表示されます。お使いのブラウザの言語設定に応じて、日本語または英語で表示されます。

---

## 2. 最初のノートを作る

ノートは「フォルダ」のようなものです。リーフ（メモ）をまとめる入れ物です。

1. 画面下の「**+📁**」ボタンをタップ
2. ノート名を入力（例：「仕事」「買い物リスト」）
3. 「作成」をタップ

ノートが作成され、中に入ります。

---

## 3. 最初のリーフを作る

リーフは実際にメモを書く「ページ」です。

1. ノートの中で、画面下の「**+📄**」ボタンをタップ
2. リーフ名を入力（例：「今日のタスク」）
3. 「作成」をタップ

エディタが開き、Markdownでメモを書けます。

---

## 4. 書いてみる

エディタにMarkdown形式でメモを書きます。

```markdown
# 今日のタスク

- [ ] 牛乳を買う
- [ ] メールを返信する
- [x] Agasteerを試す
```

書いた内容は**自動的にブラウザに保存**されます。手動で保存する必要はありません。

---

## 5. 次のステップ

### データをGitHubに保存したい場合

ブラウザのデータはキャッシュクリアで消える可能性があります。大切なメモを残すなら、GitHub連携を設定しましょう。

→ [GitHub連携セットアップ](./github-setup.md)

### 基本的な使い方を学ぶ

ノートの整理、検索、移動など、基本操作を学びましょう。

→ [基本操作](./basic-usage.md)

### スマホのホーム画面に追加したい場合

PWA対応なので、ネイティブアプリのように使えます。

**iOS（Safari）**

1. Safariで開く
2. 共有ボタン → 「ホーム画面に追加」

**Android（Chrome）**

1. Chromeで開く
2. メニュー → 「ホーム画面に追加」
