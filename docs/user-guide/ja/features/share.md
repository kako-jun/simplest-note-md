# シェア機能

ノートやリーフをURL・Markdown・QRコードで共有できます。

---

## URLをコピー

パンくずリストの「**🔗**」アイコンをタップ。

- 現在の画面のURLがクリップボードにコピーされる
- 他の人に送ったり、ブックマークに保存できる

---

## Markdownリンクをコピー

パンくずリストの「**📋**」アイコンをタップ。

コピーされる形式:

```markdown
[Agasteer > ノート1 > リーフ1](https://agasteer.llll-ll.com/note/abc/leaf/def)
```

> **💡 Tip**: 他のMarkdownドキュメントにリンクを貼りたいときに便利。

---

## QRコード

シェアメニューにリーフ内容のQRコードが表示されます。

### 使い方

1. シェアメニューを開く
2. QRコードをスマホのカメラで読み取る
3. テキストとして取得できる

### 制限

- 最大約980文字（日本語）まで
- 長すぎる場合は非表示になる

> **💡 Tip**: PCで書いた内容をスマホに転送したいときに便利。
