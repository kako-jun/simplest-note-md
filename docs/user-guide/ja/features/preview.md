# マークダウンプレビュー

リーフのMarkdownをHTMLでレンダリングして表示できます。

---

## プレビューの開き方

1. リーフを開く
2. フッターの「**👁️**」ボタンをタップ
3. もう一度タップで編集モードに戻る

---

## プレビューの特徴

- 外部リンクは新しいタブで開く
- コードブロックのシンタックスハイライト対応
- XSS対策済み（DOMPurify）

---

## 画像としてダウンロード

プレビュー表示を画像として保存できます。

1. プレビュー表示中に「**⬇️**」ボタンをタップ
2. PNG画像としてダウンロード

### 画像の仕様

- 白背景 + 20pxの余白
- スクロール全体を1枚の画像に
- ファイル名: `リーフ名.png`

---

## 画像としてシェア

1. プレビュー表示中に「**📤**」ボタンをタップ
2. シェア方法を選択

| 方法           | 説明                              |
| -------------- | --------------------------------- |
| クリップボード | 画像をコピー → 他アプリに貼り付け |
| Web Share API  | LINE、Twitterなどに直接送信       |

> **💡 Tip**: 2ペインで「左:編集、右:プレビュー」にすると、書きながら確認できます。
