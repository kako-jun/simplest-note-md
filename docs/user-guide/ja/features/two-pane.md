# 2ペイン表示

横長画面では左右2ペインで表示されます。

---

## いつ2ペインになる？

- **自動判定**: 画面の横幅 > 縦幅 のとき、自動的に2ペイン
- スマホを横向きにすると2ペインに切り替わる
- 縦長画面では常に1ペイン

---

## 左右で別々のものを開く

- 左でノート一覧、右でリーフを編集
- 左で参考資料、右で執筆中のリーフ
- 左右で異なるノートを同時に閲覧

### 同じリーフを左右で開いた場合

- **編集内容が即座に同期**（片方で入力 → もう片方にも反映）
- **スクロール位置も同期**
- 左で編集、右でプレビュー → 執筆しながら確認できる

---

## ペイン操作メニュー

ヘッダー中央の「**⇄**」ボタンをタップ。

| ボタン | 動作                 |
| ------ | -------------------- |
| →      | 左の内容を右にコピー |
| ⇄      | 左右を入れ替え       |
| ←      | 右の内容を左にコピー |

---

## 左右で異なるワールドを表示

2ペイン表示時、**左右で異なるワールド（Home/Archive）を表示**できます。

- 左ペインでHome、右ペインでArchive、といった表示が可能
- 各ペインのパンくずリストから独立してワールド切り替え可能
- アーカイブから参照しながらHomeで執筆する使い方ができる

---

## おすすめの使い方

| パターン        | 左ペイン   | 右ペイン         |
| --------------- | ---------- | ---------------- |
| 編集+プレビュー | 編集モード | プレビューモード |
| 参照しながら    | 参考リーフ | 執筆中リーフ     |
| 比較            | リーフA    | リーフB          |
| アーカイブ参照  | Archive    | Home             |

> **💡 Tip**: Vimモードなら `<Space>` でペイン切り替えできます。
