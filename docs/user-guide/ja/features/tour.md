# 初回ガイド

初めてAgasteerを使う方向けのガイド機能です。

---

## ウェルカムポップアップ

初回アクセス時に表示されるポップアップです。

### 特徴

- **言語自動検出**: ブラウザの言語設定に応じて日本語または英語で表示
- **レスポンシブ対応**: モバイルでは縦並びボタンに自動調整
- 「はじめる」をタップするとホーム画面へ

### 選択肢

| ボタン           | 説明                             |
| ---------------- | -------------------------------- |
| **はじめる**     | GitHub連携なしで開始             |
| **GitHub連携**   | 設定画面を開いてGitHub連携       |
| **セットアップ** | セットアップガイドを別タブで開く |

---

## 吹き出しガイド

初めての操作時に、ボタンの近くに吹き出しでヒントを表示します。

### 表示されるタイミング

| 条件                   | 吹き出しの内容                           |
| ---------------------- | ---------------------------------------- |
| ノートが0個のとき      | 「ここをクリックしてノートを作成」       |
| リーフが0個のとき      | 「ここをクリックしてリーフを作成」       |
| 初めて編集を行ったとき | 「GitHubに保存してデータを残しましょう」 |

### 吹き出しを消すには

- 該当のボタンをクリックする
- 吹き出し自体をクリックする（×ボタン）

一度消すと、二度と表示されません。

---

## ガイドをリセットするには

開発者コンソールで以下を実行してリロードします:

```js
const data = JSON.parse(localStorage.getItem('agasteer'))
data.state.tourShown = false
data.state.saveGuideShown = false
localStorage.setItem('agasteer', JSON.stringify(data))
```
