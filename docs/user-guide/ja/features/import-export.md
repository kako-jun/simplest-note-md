# インポート/エクスポート

データのバックアップや、他アプリからの移行ができます。

---

## ZIPエクスポート

すべてのノート・リーフをZIPファイルでダウンロード。

### 手順

1. 設定画面を開く
2. 「**ZIPエクスポート**」ボタンをタップ
3. `agasteer-export-YYYY-MM-DD.zip` がダウンロードされる

### ZIPの中身

```
agasteer-export-YYYY-MM-DD/
├── notes/
│   ├── ノート1/
│   │   ├── リーフ1.md
│   │   └── サブノート/
│   │       └── リーフ2.md
│   └── ノート2/
│       └── リーフ3.md
└── metadata.json  # 並び順、バッジ情報など
```

### 注意

- Git履歴（.git/）は含まれない
- 初回Pull完了後にエクスポート可能

> **💡 Tip**: 定期的にエクスポートしておくと、万が一のときに安心。

---

## インポート

他のメモアプリからデータを取り込めます。

### 手順

1. 設定画面を開く
2. 「**他のメモアプリからインポート**」をタップ
3. ファイルを選択

### 対応フォーマット

| フォーマット   | ファイル形式                  |
| -------------- | ----------------------------- |
| Agasteer形式   | ZIPエクスポートしたファイル   |
| SimpleNote形式 | .json（またはそれを含む.zip） |

### インポート後の挙動

- 新しいノートが作成され、その中にリーフが配置される
- 結果サマリーが表示される
- 非対応の要素（添付ファイル等）はスキップ

### 注意

- 初回Pull完了後にインポート可能
- インポート後はPushして保存することを推奨
