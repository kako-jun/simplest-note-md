# キーボードショートカット

キーボードで操作できるショートカット一覧です。

---

## 基本ショートカット

| ショートカット         | 動作                 |
| ---------------------- | -------------------- |
| **Ctrl+S** / **Cmd+S** | GitHubにPush（保存） |
| **Ctrl+クリック**      | リンクを別タブで開く |
| **中クリック**         | リンクを別タブで開く |

> **注意**: ブラウザの「ページを保存」ダイアログはCtrl+Sでオーバーライドされます。

---

## Vimモードのショートカット

Vimモードを有効にすると、以下のショートカットが使えます。

### ノーマルモード

| キー                  | 動作                        |
| --------------------- | --------------------------- |
| `i` / `a` / `o`       | 挿入モードへ                |
| `h` / `j` / `k` / `l` | カーソル移動                |
| `dd`                  | 行削除                      |
| `yy`                  | 行コピー                    |
| `p`                   | 貼り付け                    |
| `u`                   | アンドゥ                    |
| `Ctrl-r`              | リドゥ                      |
| `<Space>`             | ペイン切り替え（2ペイン時） |

### Agasteer専用コマンド

| コマンド | 動作                     |
| -------- | ------------------------ |
| `:w`     | GitHubにPush（保存）     |
| `:q`     | 親ノートへ遷移（閉じる） |
| `:wq`    | 保存して閉じる           |
