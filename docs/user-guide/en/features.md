## Toast Notifications

Results of various operations are displayed as **toast notifications** at the bottom of the screen.

| Operation      | Notification Example        |
| -------------- | --------------------------- |
| Push success   | "Saved"                     |
| Pull success   | "Data retrieved"            |
| URL copy       | "Copied to clipboard"       |
| Import success | "Imported"                  |
| Error occurred | Error details are displayed |

Notifications automatically disappear after a few seconds.
