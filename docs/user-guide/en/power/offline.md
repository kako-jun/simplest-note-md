# Offline Leaf

A local-only leaf completely independent from GitHub sync.

---

## Features

- **Unaffected by Pull/Push**: Editable even during sync
- **Stored only in browser's IndexedDB**: Not saved to GitHub
- **Not synced across devices**: Only accessible in that browser

---

## Use Cases

- Temporary notes
- Information you don't want to sync
- Working in offline environments

---
