# Self-Hosting

Agasteer is a static site, so you can host it yourself.

---

## Cloudflare Pages

1. Fork the GitHub repository
2. Cloudflare Pages → Create a project → Connect GitHub
3. Build settings:
   - Framework preset: Vite
   - Build command: `npm run build`
   - Build output directory: `dist`
4. Save and Deploy

---

## Others

Works on any service that can host static sites, including GitHub Pages, Netlify, Vercel, etc.
