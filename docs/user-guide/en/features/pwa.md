# PWA Support

Add to your phone's home screen to use like a native app.

---

## Installation

### iOS (Safari)

1. Open the demo site in Safari
2. Tap the share button (□↑)
3. Tap "**Add to Home Screen**"
4. Confirm the name and tap "Add"

### Android (Chrome)

1. Open the demo site in Chrome
2. Tap the menu (︙)
3. Tap "**Add to Home Screen**"
4. Confirm the name and tap "Add"

---

## PWA Version Features

| Item       | PWA Version            | Browser Version     |
| ---------- | ---------------------- | ------------------- |
| Launch     | Quick from home screen | Enter URL           |
| Appearance | No browser UI          | Address bar visible |
| Reload     | Use Pull button        | Reload button works |

---

## PWA Updates

App updates are automatically detected and applied.

- When a new version is available, auto-updates on next launch
- App itself stays current without manual Pull

> **Tip**: In PWA version, you can still Pull using the "⬇️" button in the header. No reload button needed.
