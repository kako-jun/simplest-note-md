# Share Feature

Share notes and leaves via URL, Markdown, or QR code.

---

## How to Open Share Menu

Tap the "**ðŸ“¤**" icon in the breadcrumb to open the share menu.

---

## Copy URL

Tap "**Copy URL**" in the share menu.

- Current screen URL is copied to clipboard
- Send to others or save as a bookmark

---

## Copy Markdown

Tap "**Copy Markdown**" in the share menu.

- Entire leaf's Markdown text is copied
- If text is selected, only the selection is copied

> **Tip**: Useful when pasting into other apps.

---

## Share as Image

Tap "**Share Image**" in the share menu (Web Share API supported devices only).

- Send directly to LINE, Twitter, etc.
- If text is selected, only the selection is converted to image

---

## QR Code

A QR code of the leaf content is displayed in the share menu.

- If text is selected, only the selection is encoded in QR
- If nothing is selected, the entire leaf is encoded

### How to Use

1. (If needed) Select the part you want to transfer
2. Open share menu
3. Scan the QR code with your phone's camera
4. Retrieve as text

### Limitations

- Maximum approximately 980 characters (Japanese)
- Hidden if too long

> **Tip**: To transfer just part of a long text, select that portion before opening the share menu.
