# Settings Storage Locations

A list of where each setting is stored.

---

## Overview

| Setting Item     | Storage Location            |
| ---------------- | --------------------------- |
| Theme            | LocalStorage + GitHub       |
| Language         | LocalStorage + GitHub       |
| Lined Mode       | LocalStorage                |
| Vim Mode         | LocalStorage                |
| Custom Font      | IndexedDB (browser only)    |
| Background Image | IndexedDB (browser only)    |
| GitHub Token     | LocalStorage (browser only) |

---

## Notes

- **LocalStorage + GitHub**: Synced across devices
- **LocalStorage**: This browser only
- **IndexedDB**: This browser only (for large data)
