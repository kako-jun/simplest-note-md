# Custom Background Image

Set different background images for left and right panes.

---

## Supported Formats

- `.jpg` / `.jpeg`
- `.png`
- `.webp`
- `.gif`

---

## How to Upload

1. Open settings
2. Tap "Select Image" button (separate for left/right pane)
3. Choose an image file
4. Applied immediately

---

## Opacity Adjustment

When a background image is set, an "Opacity" slider appears.

- **Range**: 10% - 50%
- **Default**: 10%
- Changes apply in real-time as you move the slider

> **Tip**: If the background is too faint, move the slider right to increase opacity.

---

## Remove

Use the "Clear" button to remove.

---

## Notes

- Images are stored in browser's IndexedDB
- Not saved to GitHub
- Different images can be set for left and right panes
