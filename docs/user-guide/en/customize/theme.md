# Theme

Choose from 6 themes.

---

## Theme List

| Theme          | Description                 |
| -------------- | --------------------------- |
| **yomi**       | Calm yellow tones (default) |
| **campus**     | Warm campus notebook style  |
| **greenboard** | Chalkboard-style dark theme |
| **whiteboard** | Bright whiteboard style     |
| **dotsD**      | Dot pattern dark theme      |
| **dotsF**      | Dot pattern light theme     |

---

## How to Change

1. Open settings (⚙️ in top-right header)
2. Tap a theme button
3. Changes apply immediately

---

## App Icon

The app icon is displayed in the top-left header (only when tool name is default "Agasteer").

- Drawn with **accent color** matching the theme
- Icon color changes when you change theme
