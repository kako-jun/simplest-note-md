# Quick Start

> Get started with Agasteer in 5 minutes.

---

## 1. Open the Demo Site

Just visit **[https://agasteer.llll-ll.com](https://agasteer.llll-ll.com)** to start using it immediately.

- No installation required
- No account registration required
- Just open in your browser

On your first visit, a **welcome popup** will appear. It will display in Japanese or English depending on your browser's language settings.

---

## 2. Set Up GitHub Integration

**Important**: You need to set up GitHub integration before you can create notes or leaves.

1. Tap the **gear icon** (Settings) in the top right corner
2. In the **GitHub Integration** section, enter your repository name and token
3. Tap the "Pull" button to verify the connection

For detailed instructions → [GitHub Integration Setup](./github-setup.md)

---

## 3. Create Your First Note

A note is like a "folder." It's a container for organizing leaves (memos).

1. Tap the "**+folder**" button at the bottom of the screen
2. Enter a note name (e.g., "Work", "Shopping List")
3. Tap "Create"

The note is created and you'll enter it.

---

## 4. Create Your First Leaf

A leaf is the actual "page" where you write your memos.

1. Inside a note, tap the "**+file**" button at the bottom of the screen
2. Enter a leaf name (e.g., "Today's Tasks")
3. Tap "Create"

The editor opens and you can write in Markdown.

---

## 5. Try Writing

Write your memo in Markdown format in the editor.

```markdown
# Today's Tasks

- [ ] Buy milk
- [ ] Reply to emails
- [x] Try Agasteer
```

Your content is **automatically saved to your browser**. To save to GitHub, tap the **save button** in the top right corner.
