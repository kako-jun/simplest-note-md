## 2ペイン表示

### アスペクト比判定

画面のアスペクト比（横 > 縦）で2ペイン表示を自動切替。`window.innerWidth > window.innerHeight`で判定し、resizeイベントで更新します。

### レスポンシブレイアウト

`.content-wrapper`にCSS Gridを使用し、`single-pane`クラスで1カラム/2カラムを切り替え。右ペインは`.hidden`クラスで非表示にします。

### 動作

- **スマホ縦向き**: 1ペイン表示
- **スマホ横向き**: 2ペイン表示
- **PC横長画面**: 2ペイン表示
- **画面回転時**: 自動的に切り替わる

### ペイン操作メニュー

2ペイン表示時、ヘッダー中央に表示されるボタン（⇄）をタップするとメニューが表示されます。

- **→**: 左ペインの状態を右ペインにコピー
- **⇄**: 左右ペインの状態を入れ替え
- **←**: 右ペインの状態を左ペインにコピー

**関連ファイル**: `Header.svelte`, `App.svelte`
