## バッジ機能（アイコン＋色パレット）

### 概要

ノートとリーフのカード右上にバッジを表示する機能。アイコンと色を自由に組み合わせて視覚的に区別できます。

### データ構造

Note型とLeaf型に以下のオプションフィールドを追加：

| フィールド | 型     | 説明                                  |
| ---------- | ------ | ------------------------------------- |
| badgeIcon  | string | アイコン識別子（例: 'star', 'heart'） |
| badgeColor | string | カラーコード（例: '#ff6b6b'）         |

### UI実装

#### バッジ表示

- バッジ未設定時: 「+」マークを表示
- バッジ設定時: アイコンを指定色で表示
- クリックでバッジピッカーを開く

#### アイコン選択UI（5×5グリッド）

利用可能なアイコン（25種類）:

- スター、ハート、チェック、フラグ、ブックマーク
- 電球、ピン、ベル、時計、カレンダー
- その他フォントアイコン

#### 色選択UI（5色パレット）

| 色  | カラーコード |
| --- | ------------ |
| 赤  | #ff6b6b      |
| 黄  | #ffd93d      |
| 緑  | #6bcb77      |
| 青  | #4d96ff      |
| 紫  | #9b59b6      |

### 保存先

- **IndexedDB**: ノート/リーフのフィールドとして保存
- **GitHub**: `metadata.json`内の各ノート/リーフエントリに保存
