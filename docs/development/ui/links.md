## タイトルリンク化

### 概要

ヘッダーのタイトルをクリック可能なリンクに変更し、Ctrl+クリックや中クリックで別タブを開けるようにする機能。

### 実装

ボタンを`<a>`タグに変更し、修飾キーなしの通常クリックのみ`e.preventDefault()`で既存動作を維持。

### 動作

| 操作                           | 動作                                       |
| ------------------------------ | ------------------------------------------ |
| **通常クリック**               | ホーム画面に遷移（既存動作）               |
| **Ctrl+クリック（Win/Linux）** | 新しいタブでホームを開く                   |
| **Cmd+クリック（Mac）**        | 新しいタブでホームを開く                   |
| **中クリック**                 | 新しいタブでホームを開く                   |
| **Shift+クリック**             | 新しいウィンドウでホームを開く             |
| **右クリック**                 | コンテキストメニュー（新しいタブで開く等） |

### ユースケース

- 複数タブでアプリを開く
- 左右のペインで別々のノートを開く
- ブラウザの「戻る」ボタンで履歴を辿る

## 外部リンク

### 概要

アプリ内から外部URL（GitHub、ドキュメント等）を開く際の実装パターン。

### 推奨パターン

PWAでも外部ブラウザで開くよう、シンプルな `<a>` タグを使用：

```svelte
<a href="https://example.com" target="_blank" rel="noopener noreferrer" on:click|stopPropagation>
  リンクテキスト
</a>
```

### 仕様

| 属性                        | 説明                                           |
| --------------------------- | ---------------------------------------------- |
| `target="_blank"`           | 新しいタブ/ウィンドウで開く                    |
| `rel="noopener noreferrer"` | セキュリティ対策（`window.opener` を無効化）   |
| `on:click\|stopPropagation` | 親要素のクリックイベントが発火しないようにする |

### 使用箇所

- GitHub設定のヘルプアイコン
- Offline/Priorityカードのヘルプアイコン
- 設定画面の動画リンク
- プレビュー内のリンク

## GitHub設定ヘルプアイコン

### 概要

GitHub設定の入力欄（リポジトリ名・トークン）に「？」アイコンを追加し、クリックで外部ドキュメントを開く。

### 実装

ラベルの横に「？」アイコンを配置し、`<a target="_blank">` で GitHub 上のユーザーガイドにリンク。

### デザイン

| 項目   | 値                               |
| ------ | -------------------------------- |
| 位置   | ラベルの右横                     |
| サイズ | 18×18px                          |
| 色     | アクセントカラー（opacity: 0.7） |
| ホバー | opacity: 1、scale: 1.1           |

### リンク先

| 項目           | リンク先                                   |
| -------------- | ------------------------------------------ |
| リポジトリ名   | `docs/user-guide/{lang}/github-setup.md`   |
| トークン       | 同上（アンカーでセクションにジャンプ）     |
| Offlineカード  | `docs/user-guide/{lang}/power/offline.md`  |
| Priorityカード | `docs/user-guide/{lang}/power/priority.md` |
